Gems

Currently used

  • Devise: a flexible authentication solution for Rails based on Warde
  • Gritter: easily add Growl-like notifications to your application using a jQuery plugin called 'gritter'
  • FriendlyID: slugging and permalink plugins for Active Record. It lets you create pretty URLs and work with human-friendly strings as if they were numeric ids.
  • Kaminari: a Scope & Engine based, clean, powerful, agnostic, customizable and sophisticated paginator for Rails 3+
  • Dragonfly: on-the-fly processing - suitable for image uploading in Rails, Sinatra and much more
  • Pundit: object oriented authorization for Rails applications
  • Active Admin: the administration framework
  • Ancestry: organise ActiveRecord model into a tree structure
  • OmniAuth: a flexible authentication system utilizing Rack middleware
  • Breakman: an open source vulnerability scanner specifically designed for Ruby on Rails applications. It statically analyzes Rails application code to find security issues at any stage of development
  • Prawn: fast, Nimble PDF Writer for Ruby
  • MailBoxer: a private message system
  • Better Errors: Better error page for Rack apps
  • Guard: a command line tool to easily handle events on file system modifications
  • Annotate: Add a comment summarizing the current schema to the top of each ActiveRecord model source file
  • Flay: analyzes code for structural similarities. Differences in literal values, variable, class, method names, whitespace, programming style, braces vs do/end, etc are all ignored
  • Table Print: It shows objects in nicely formatted columns for easy reading. It even lets you nest other tables of related objects, contextualizing data across tables.
  • Local Tunnel: Expose your local web server to the Internet.
  • Quiet Assets: Turn off rails assets log.
  • Rack Mini Profiler: Profiling toolkit for Rack applications with Rails integration. Client Side profiling, DB profiling and Server profiling.
  • Public Activity: Easy activity tracking for your ActiveRecord models. Provides Activity model with details about actions performed by your users, like adding comments, responding etc.
  • Jammit: an industrial strength asset packaging library for Rails, providing both the CSS and JavaScript concatenation and compression that you'd expect, as well as YUI Compressor and Closure Compiler compatibility, ahead-of-time gzipping, built-in JavaScript template support, and optional Data-URI / MHTML image embedding.

Suggestive Gems

  • Forem: the best Rails 3 and Rails 4 forum engine. Ever.
  • Flutie: a starting point for personal discovery
  • Capybara: an integration testing tool for rack based web applications. It simulates how a user would interact with a website
  • Active Merchant: a simple payment abstraction library used in and sponsored by Shopify.
  • Discourse: a platform for community discussion.
  • Koala: a lightweight, flexible library for Facebook with support for OAuth authentication, the Graph and REST APIs, realtime updates, and test users.
  • Instagram Ruby Gem: The official gem for the Instagram API
  • Octokit: Ruby toolkit for the GitHub API
  • AWS SDK for Ruby: The official AWS SDK for Ruby.
  • Youtube it: An object-oriented Ruby wrapper for the YouTube GData API
